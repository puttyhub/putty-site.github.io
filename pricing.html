Scope of Work

The scope of this project includes:

  • Client Hardware & System Assessment: Evaluate current servers and hardware, assess employee workstations, and provide recommendations.
  • Comfy Setup: Install necessary nodes and models, make the system accessible via reverse proxy, create basic workflows, and set up FileBrowser for shared file access.
  • Mesh Network Configuration: Set up a mesh network for team connectivity and guide team members through SSH setup.
  • Reverse Proxy Configuration: Coordinate with team members via email to establish secure access points.
  • Security & User Access: Implement security best practices and manage user access controls.
  • Dashboard & Admin Access: Create a dashboard portal for resource monitoring and enable admin controls.
  • Coordination & Onboarding: Assist team members during setup and provide guidance and documentation.
